# BarcodeScanner
## BarcodeScannerとは？
Barcode Scannerとは、Kotlinで書かれたAndroid用のバーコードをスキャンするアプリです。
ScanResult Viewerと合わせて使用する事で、スキャン結果の確認、編集等が可能になります。

### 機能
+ 写真撮影
+ 写真内からバーコード検知、検知エリアに枠線表示(複数対応)
+ バーコード枠をタップする事で、バーコード、QRコードを読み取り保存
+ 写真保存ボタンをタップする事で、読み取りは実行せず、写真として保存
+ 共有ストレージ内のアプリフォルダに、撮影した画像(jpg)と認識結果(txt)が保存される

### 制約
+ APIレベル29以降 (Android10)

### 注意事項
+ 斜めに撮影するとバーコード、QRコード等が読めない事があります。可能な限り真っすぐに撮影してください。
