# BarcodeScanner
## BarcodeScannerとは？
Barcode Scannerとは、Kotlinで書かれたAndroid用のバーコードをスキャンするアプリです。
特別珍しいものではなく、自社内にて要望のあったツールを、AIバイブコーディングの練習材料として開発した物になります。

### 機能
+ 写真撮影
+ 写真内からバーコード検知、検知エリアに枠線表示
+ スキャン機能によりバーコードを認識、結果表示
+ 共有ストレージ内のアプリフォルダに、撮影した画像(jpg)と認識結果(txt)を保存

### 制約
+ APIレベル29以降 (Android10)

### 注意事項
+ 斜めに撮影するとバーコード、QRコード等が読めない事があります。可能な限り真っすぐに撮影してください。

### 今後の予定
+ 現在、スキャン結果は共有ストレージ内に保存され、取り出すには、スキャン毎に個別に作成されたテキストファイルを開く必要があり、数が多いと手間が掛かる。一覧で取得できる等、ここをもう少し便利にしたい。
+ ユニットテストコードの追加
